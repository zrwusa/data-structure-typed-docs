SkipList

Defined in: data-structures/linked-list/skip-linked-list.ts:49

SkipList — a probabilistic sorted key-value container.

API mirrors TreeMap: users can swap TreeMap ↔ SkipList with zero code changes. Reference: Java ConcurrentSkipListMap (NavigableMap interface).
